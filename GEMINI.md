# Правила проекта: Формула цены МинВаты

## Технологический стек
- **Расчеты:** Приоритет отдается Python-модулям `minwool/engine.py`, `minwool/gui.py`, `minwool/io.py` (через совместимый entrypoint `minwool_engine.py`).
- **Библиотеки:** `pandas`, `openpyxl`, `xlsxwriter`.

## Логика расчетов (Mandatory)
1. **Упаковка:** 
   - Расчет пленки вести через погонные метры (ширина пленки 1.4 м): `((2 * H_pack + 2 * W_pack) / 1000) * Цена_п.м`.
   - Стоимость паллетной упаковки: `Пленка_пачки * Пачек_на_паллете + Худ + Стрейч`.
   - Распределенная стоимость пачки: `Стоимость_упаковки_паллета / Пачек_на_паллете`.
2. **Оптимизация:** Количество плит в пачке должно стремиться к высоте 600 мм, если не указано иное.
3. **Версионирование:** При изменении структуры Excel увеличивать версию (v2h -> v2i и т.д.).

## Команды
- При изменении входящих цен (газ, смола, пленка) — обновлять словарь `config` в `minwool_engine.py` и перегенерировать Excel.
